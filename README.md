# Sistema de Ordem de Serviço

Este projeto é um sistema simples em C++ para gerenciamento de ordens de serviço.

## Funcionalidades
- Criar ordem de serviço
- Listar ordens
- Editar ordens
- Deletar ordens

## Tecnologias
- C++
- STL (vector)

## Observações
Projeto desenvolvido com foco em prática de lógica de programação
